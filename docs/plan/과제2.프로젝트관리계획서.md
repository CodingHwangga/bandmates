## 1. 프로젝트 개요

### 1.1 프로젝트 목적
BandMate는 밴드 멤버들이 합주 일정, 곡 목록, 연습 자료를 효율적으로 관리할 수 있는 웹 플랫폼을 개발하는 것을 목표로 한다.

### 1.2 프로젝트 배경
- 기존 밴드 활동에서는 일정 조율, 악보 공유, 녹음 파일 관리가 분산되어 비효율적
- 카카오톡, 구글 드라이브, 캘린더 등 여러 도구를 동시에 사용해야 하는 불편함
- 통합된 밴드 관리 솔루션의 필요성 대두

### 1.3 프로젝트 범위
- **포함 사항**:
  - 밴드 생성 및 멤버 관리 기능
  - 합주 일정 캘린더 관리
  - 곡 목록 및 셋리스트 관리
  - 연습 자료(악보, 탭, MR 파일) 공유
  - 합주 녹음 파일 저장 및 재생
  
- **제외 사항**:
  - 실시간 영상 통화 기능
  - 실시간 협업 편곡 기능
  - 모바일 네이티브 앱 (웹 기반만 제공)
